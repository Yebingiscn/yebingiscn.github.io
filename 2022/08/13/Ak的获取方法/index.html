Ak 的获取方法

最近在做百度地图 API 相关的项目，需要用到获取安卓的 AK，简单来说就是一种密钥。
但是获取的时候踩了些坑，在这里书写下怎么避免。

  • 首先要注意自己的.android 路径，像我的其实放在 D 盘而不是和百度给的实例一样放在 C 盘
  • 要带上 jdk 路径和-list -v -keystore debug.keystore。这里如果按照百度上来则是**巨坑
  • 密码是 android
